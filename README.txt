Инструкции за пускане програмите върху базовите тестове.

Windows
-------

1. Компилирайте програмите с любимата си среда за разработка.
2. Копирайте изпълнимите файлове prog*.exe в папката с тестовете.
3. Пуснете test.bat с двойно кликване.
4. Вижте резултатите

Linux
-----

1. Осигурете си, че имате инсталиран компилатор GCC (g++)
2. Копирайте изходните файлове prog*.cpp в папката с тестовете
3. Отворете терминала
4. cd <директорията-с-тестовете>
5. chmod +x test.sh
6. ./test.sh
7. Вижте резултатите.
   Легенда:
     * OK = тестът минава
     * NA = Not Available = няма програма
     * WA = Wrong Answer = програмата връща грешен отговор
     * CE = Compile Error = грешка при компилиране
     * RE = Runtime Error = грешка при изпълнение
     * TO = Time-Out = програмата отнема твърде много време за изпълнение
     * OL = Output Limit = програмата извежда твърде много на стандартния изход
